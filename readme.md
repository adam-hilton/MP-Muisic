# Midi Hand Tracker

Midi Hand Tracker is a simple script that uses Mediapipe, OpenCV, and Mido to turn your hand movements into MIDI data.

## Dependencies & Usage

Libraries
- Mediapipe
- OpenCV
- Mido

If running headless while connected to a monitor:
execute 
``export DISPLAY=:0``
in terminal prior to running
